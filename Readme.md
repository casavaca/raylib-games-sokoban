# How to build and run:

+ Debug build:
  `mkdir build && cd build && cmake .. -DCMAKE_BUILD_TYPE=DEBUG && make && ./sokoban`

+ Release build:
  `mkdir build && cd build && cmake .. && make && ./sokoban`
